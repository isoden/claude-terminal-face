ramp t
ramp t は hook（claude-face-ramp.sh）が prev→current の色を送り終えるまでの秒数（RAMP_SEC=1.26s）。0 で prev 色、1.26 で current 色。シェーダーは iCurrentCursorColor だけを見るので、ここで中間色を合成して実機の見え方を再現する。
variant lock
各フェーズが持つ表情（think は 4、他は 3）のうち 1 つに固定する（呼吸・ドリフト・瞬き等の動きはそのまま）。ロード時に pickVar() の定義を差し替えて実現しているので、実機のシェーダーには影響しない。err はバリアントを持たないので無影響。

tweak (@tweak 定数)

シェーダー内で // @tweak <min> <max> が付いた const float を uniform 化して直接触る。気持ちいい値が決まったら「diff を出力」でコンソールに patch を出し、GLSL 本体へ反映する（GUI は探索用で本番機能ではない）。
